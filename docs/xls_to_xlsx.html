[サンプルページ].xlsを.xlsxへ一括置換したい

.xlsファイルを.xlsxに一括変換したい。 ※ サンプルページです

利用シーンと目的

大学で使用しているWebアプリが古くて.xls出力にしか対応していない。
でも入力ファイルは.xlsx形式しか受け付けてくれない。
→複数の.xlsファイルを.xlsxファイルへ一括置換したい

実装の考え方

結論だけ言えば、pandasで読み込んだxlsをxlsxで出力すれば良い。pythonにはExcelを扱う方法が複数用意されている。(記事を参照しつつ紹介)

コード

実際のコードがこちら。クラス内メソッドをコピペしただけなので粗が残っている。


  import pandas as pd
  from pathlib import Path

  def xls_to_xlsx(self):
      # 対象フォルダ内のxlsファイルを読み込む
      for xls_file in self.input_path.glob("./*.xls"):
          df_xls = pd.read_excel(xls_file)

          print(f"{xls_file.name}を{xls_file.stem}.xlsxへ変換します")
          df_xls.to_excel(self.output_path.joinpath(xls_file.stem + ".xlsx"), index=False)
      return 0

テーブル
